# timer

## What this project does

Timer is a time setting app for presenters and judges.
This app support pip mode, full screen setting.
This app support time save, reset, preset, alarm feature.

## Tech stack

- Runtime: Node.js v18+
- Bundler: Vite
- Library: React v18+
- Language: TypeScript
- Styling: Tailwind CSS v3
- Linter: ESLint v9+

## Run commands

- Vite Setting: npm create vite@latest
- Install: npm install
- Run: npm run dev
- Build: npm run build

## Conventions

- Use TypeScript strict mode
- Handle all errors gracefully with user-friendly messages
